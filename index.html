Data Visualization Assignment

This project involves creating visualizations using D3, based on data resulting from dimensionality reduction and clustering applied to descriptions of Bob Ross paintings. Each entry in the dataset represents an individual painting, with binary attributes (0 or 1) indicating the presence of specific terrain features, whether natural or man-made. These attributes are extracted from Bob Ross's painting descriptions and are visually explored through techniques like Principal Component Analysis (PCA) and k-means.

The visualizations include:

  1. PCA Scatter Plot: Shows the 2D projection of data after PCA. X-axis represents the first principal component, and the Y-axis represents the second principal component.
  2. Divergent Bar Plot for PCA Loadings: Displays two bars for each attribute, one for each principal component, showing the loadings. Divergent bars indicate the direction (positive/negative) of the loading.
  3. Cluster Heatmap: Visualizes k-means results with attribute labels on the Y-axis, landform feature names on the X-axis, and colors representing the sum of attribute values for a given label and attribute combination.

Note: To generate the data and create the chart, you need to run the server.py file.

To run ccPCA, you must select only one cluster (by clicking on the circles next to the heatmap).
